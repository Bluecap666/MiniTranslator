### 使用技巧

1. **窗口操作**
   - 拖动顶部的紫色标题栏可以移动窗口
   - 拖动右下角的调整手柄可以改变窗口大小
   - 点击 ⚙️ 图标打开设置
   - 点击 − 最小化窗口
   - 点击 × 关闭应用

2. **翻译功能**
   - 在上方输入框输入文本
   - 选择翻译服务（默认 Google 翻译）
   - 等待 1 秒自动翻译，或点击"翻译"按钮
   - 按 Ctrl+Enter 也可以快速翻译

3. **配置 API**
   - 点击齿轮图标 ⚙️ 打开设置
   - 选择不同的翻译服务
   - Google 翻译无需配置即可使用
   - 百度翻译和有道翻译需要填写 API 密钥

### API 获取方式

#### 百度翻译
1. 访问：https://fanyi-api.baidu.com/
2. 注册登录
3. 进入"管理控制台"
4. 开通通用翻译 API
5. 获取 APP ID 和密钥

#### 有道翻译
1. 访问：https://ai.youdao.com/
2. 注册登录
3. 创建应用
4. 获取应用 ID 和密钥

### 注意事项
- Google 翻译使用的是免费接口，建议优先使用
- 如需更稳定的服务，可配置百度或有道翻译 API
- 窗口会自动置顶，方便在使用其他应用时随时查看翻译结果
