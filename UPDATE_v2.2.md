# 🎉 更新日志 - v2.2 AI 翻译版

## ✨ 重大更新

### 1. 🤖 新增 AI 翻译服务

现已集成两大主流 AI 大模型翻译服务！

#### 通义千问（Qwen）
- **提供商**: 阿里云 DashScope
- **特点**: 
  - 基于千亿参数大模型
  - 支持上下文理解
  - 智能语义分析
  - 翻译质量接近人工
- **免费额度**: 新用户赠送额度
- **获取方式**: https://dashscope.console.aliyun.com/

#### 文心一言（Ernie）
- **提供商**: 百度智能云
- **特点**:
  - 百度大模型技术
  - 中文翻译质量优秀
  - 多语言互译
  - 文化背景理解
- **免费额度**: 每日免费调用次数
- **获取方式**: https://console.bce.baidu.com/ai/

### 2. 🔧 问题修复

#### 修复配置窗口按钮无响应
**问题描述**: 点击"测试连接"和"保存配置"按钮没有反应

**修复方案**:
- 优化了 IPC 通信机制
- 添加了详细的日志输出
- 增强了错误处理
- 确保按钮事件正确绑定

**验证方法**:
```
1. 打开配置窗口
2. 填写任意 API 配置
3. 点击"测试连接"
4. 查看控制台日志和测试结果
5. 点击"保存配置"
6. 确认保存成功提示
```

#### 默认翻译服务改为百度翻译
**原因**:
- 百度翻译国内访问更稳定
- QPS 限制较宽松
- 适合日常使用
- 作为故障切换的首选备选

**修改位置**:
- config-window.html: 下拉框默认选项
- main.js: 初始配置默认值
- config.js: 保存配置默认值

#### 配置窗口关闭问题
**问题描述**: 关闭窗口后资源未完全释放

**修复方案**:
- 添加窗口关闭日志
- 手动清空 configWindow 引用
- 添加 config-updated 事件处理
- 确保窗口正确关闭和资源释放

## 💡 使用指南

### AI 翻译使用步骤

#### 1. 获取 API Key

**通义千问**:
```
1. 访问：https://dashscope.console.aliyun.com/
2. 注册/登录阿里云账号
3. 开通 DashScope 服务
4. 创建 API Key
5. 复制 Key 到配置窗口
```

**文心一言**:
```
1. 访问：https://console.bce.baidu.com/ai/
2. 注册/登录百度账号
3. 进入文心一言控制台
4. 创建应用获取 Key
5. 复制 Key 到配置窗口
```

#### 2. 配置 AI 服务
```
1. 点击设置图标 ⚙️
2. 选择 "🤖 通义千问（AI）" 或 "🤖 文心一言（AI）"
3. 粘贴 API Key
4. 点击"测试连接"验证
5. 保存配置
```

#### 3. 使用 AI 翻译
```
1. 主窗口选择对应的 AI 服务
2. 输入要翻译的文本
3. 等待 AI 智能翻译
4. 查看高质量的翻译结果
```

### 错误处理增强

```javascript
// 添加详细日志
console.log('保存配置按钮被点击');
console.log('准备保存配置:', JSON.stringify(config, null, 2));
console.log('保存结果:', result);

// 增强错误捕获
try {
  const result = await ipcRenderer.invoke('save-config', config);
  if (result.success) {
    alert('✓ 配置已保存！');
  } else {
    alert('✗ 保存失败：' + result.error);
  }
} catch (error) {
  console.error('保存配置出错:', error);
  alert('✗ 保存失败：' + error.message);
}
```
