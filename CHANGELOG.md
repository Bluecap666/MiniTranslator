# 更新日志 - v2.0

## 🎉 重大更新

### ✨ 新增翻译服务

现已支持 **6 种** 翻译服务，满足各种场景需求：

1. **Google 翻译** ⭐ 推荐
   - 无需配置，开箱即用
   - 免费使用
   - 支持全球多种语言

2. **彩云小译**
   - 高质量中英文互译
   - 需要 API Token
   - 获取方式：https://dashboard.caiyuntrans.com

3. **DeepL**
   - 业界公认的高质量翻译
   - 需要 API Key
   - 获取方式：https://www.deepl.com/pro-api

4. **百度翻译**
   - 国内稳定可靠
   - 需要 APP ID 和密钥
   - 获取方式：https://fanyi-api.baidu.com/

5. **有道翻译**
   - 网易出品，质量优秀
   - 需要应用 ID 和密钥
   - 获取方式：https://ai.youdao.com/

6. **腾讯翻译**
   - 腾讯云出品，稳定快速
   - 需要 Secret ID 和 Secret Key
   - 获取方式：腾讯云控制台

### 🔄 智能故障切换

- **自动重试机制**：当首选翻译服务失败时，自动按顺序尝试其他服务
- **优先级排序**：Google → 彩云 → DeepL → 百度 → 有道 → 腾讯
- **无缝切换**：用户无需手动干预，确保翻译成功率
- **详细日志**：在控制台显示每次尝试的服务和结果

### ⌨️ 全局快捷键

- **默认快捷键**：`Ctrl + Shift + X` (Mac: `Command + Shift + X`)
- **功能**：显示/隐藏翻译窗口
- **优势**：
  - 快速呼出，不占用任务栏
  - 专注工作时一键显示/隐藏
  - 提高生产力

## 💡 使用建议

### 推荐配置
1. **日常使用**：Google 翻译（免费、无需配置）
2. **专业场景**：DeepL（高质量）
3. **备选方案**：配置 2-3 个服务，确保稳定性

### 快捷键技巧
- 工作中需要频繁翻译：记住 `Ctrl+Shift+X`
- 窗口隐藏后，任务栏不会显示图标
- 再次按下快捷键立即显示

### 故障切换
- 默认会自动尝试所有可用服务
- 如果某个服务持续失败，检查 API 配置
- 查看控制台日志了解详细信息
