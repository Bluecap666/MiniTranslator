## 🚀 如何使用 AI 翻译

### 步骤 1: 配置 AI 服务 API

1. **打开配置窗口** - 点击 ⚙️ 图标
2. **滚动到底部** - 找到 AI 翻译配置区域
3. **填写 API Key**:

**通义千问**:
```
🤖 通义千问（AI 翻译）
API Key: [sk-xxxxxxxxxxxxxxxx]
获取方式：https://dashscope.console.aliyun.com/
特点：基于大模型的智能翻译，支持上下文理解
```

**文心一言**:
```
🤖 文心一言（AI 翻译）
API Key: [xxxxxxxxxxxxxxxx]
获取方式：https://console.bce.baidu.com/ai/
特点：百度大模型，中文翻译质量优秀
```

4. **点击"测试连接"** - 验证 API 是否有效
5. **点击"保存配置"** - 保存设置

### 步骤 2: 在主窗口使用 AI 翻译

1. **返回主窗口**
2. **打开下拉框** - 点击服务选择框
3. **选择 AI 服务**:
   - 🤖 通义千问（AI）
   - 🤖 文心一言（AI）
4. **输入要翻译的文本**
5. **点击"翻译"按钮**
6. **查看 AI 翻译结果**

## 💡 AI 翻译 vs 传统翻译

### AI 翻译的优势

✅ **上下文理解**
- 能理解整句甚至整段的含义
- 翻译更自然流畅
- 适合长文本、复杂文本

✅ **智能处理**
- 支持多种语言风格
- 能处理专业术语
- 翻译质量接近人工

✅ **持续进化**
- 模型不断训练优化
- 适应新的表达方式

### AI 翻译的特点

⚠️ **需要 API Key**
- 需要到相应平台申请
- 通常有免费额度
- 超出后可能收费

⚠️ **速度稍慢**
- 处理复杂度高于传统翻译
- 网络延迟稍大
- 但质量更高

⚠️ **适合场景**
- 重要文档翻译
- 专业内容翻译
- 需要高质量的场景

### 传统翻译的特点

✅ **快速**
- 即时响应
- 适合大量短文本

✅ **免费**
- Google 完全免费
- 其他服务也有免费额度

✅ **稳定**
- 技术成熟
- 服务可靠

**调用流程**:
```

→ 点击翻译
→ renderer.js 发送 IPC 请求
→ main.js 调用对应的 AI 翻译函数
→ 发送 HTTPS 请求到 AI 平台 API
→ 接收并解析结果
→ 返回翻译结果到前端
```
