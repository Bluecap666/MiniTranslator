## ✨ 功能特点

## 📖 使用说明

### 🎯 基本使用流程

1. **输入文本**
   - 在上方输入框输入要翻译的文本
   - 支持中英文、日文、韩文等多种语言

2. **执行翻译**
   - 点击「翻译」按钮
   - 或使用快捷键 `Ctrl + Enter`
   - 系统自动调用所有已配置的翻译服务

3. **查看结果**
   - 翻译结果以列表形式展示
   - 每个平台的结果独立显示
   - 格式：`平台名称：翻译结果`

4. **清空内容**
   - 点击「🗑️ 清空」按钮
   - 一键清空输入和输出区域

#### 配置步骤

**第 1 步：打开配置窗口**
- 点击主窗口右上角的 ⚙️ 齿轮图标

**第 2 步：填写 API 信息**
- 选择需要配置的服务
- 填写对应的 API Key / Secret
- 点击「测试」按钮验证可用性
- 显示「✓ 可用」表示配置成功

**第 3 步：保存配置**
- 点击「保存配置」按钮
- 配置信息保存到 `config.json`
- 关闭窗口返回主界面

## 💡 常见问题 FAQ

### Q1: 为什么翻译结果为空？
**A**: 可能是以下原因：
- 未配置任何 API 密钥（Google 除外）
- API 密钥无效或过期
- 网络连接问题
- 请求超时（超过 8 秒）

**解决方案**：
1. 打开配置窗口检查已配置的服务
2. 点击「测试」按钮验证 API 可用性
3. 检查网络连接是否正常

### Q2: 配置文件在哪里？
**A**: 
- **便携版**: `config.json` 在 exe 同目录下
- **源码版**: `config.json` 在项目根目录

### Q3: 如何备份配置？
**A**: 
直接复制 `config.json` 文件即可。重装系统后放回原位置就能恢复配置。

### Q4: 可以只使用某一个翻译服务吗？
**A**: 
可以！只需要配置该服务的 API 密钥，其他服务留空即可。翻译时会自动跳过未配置的服务。

### Q5: 为什么有些服务测试失败？
**A**: 
- API 密钥填写错误
- 账户余额不足
- 网络被防火墙拦截
- API 服务暂时不可用

### Q6: 如何让窗口不总是置顶？
**A**: 
当前版本默认始终置顶，如需修改可以在 `main.js` 中调整 `alwaysOnTop` 参数。
