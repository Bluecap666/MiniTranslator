## 🔍 常见问题排查

### 问题 1: 点击关闭后托盘图标不出现

**可能原因**:
1. 代码未正确执行
2. 托盘创建失败

**检查方法**:
```powershell
# 查看控制台输出
# 应该看到：[Main] 系统托盘已创建
# 应该看到：[Main] 窗口已隐藏到托盘
```

**解决方法**:
- 重启程序
- 检查控制台是否有错误信息
- 确认代码已更新

---

### 问题 2: 托盘图标是空白的

**原因**: 没有 `favicon.ico` 文件

**影响**: 
- ⚠️ 不影响功能
- ⚠️ 只是图标不好看

**解决方法**（可选）:
1. 准备一个 16x16 或 32x32 的 `.ico` 文件
2. 命名为 `favicon.ico`
3. 放到项目根目录
4. 重启程序

---

### 问题 3: 快捷键没反应

**可能原因**:
1. 快捷键被其他软件占用
2. 窗口对象为空

**检查方法**:
```javascript
// 在 main.js 中添加调试日志
globalShortcut.register('CommandOrControl+Shift+X', () => {
  console.log('[Main] 快捷键被触发'); // ← 添加这行
  if (mainWindow) {
    console.log('[Main] mainWindow 存在');
    // ...
  } else {
    console.log('[Main] mainWindow 不存在');
  }
});
```

**解决方法**:
- 关闭可能占用该快捷键的软件
- 或者修改为其他快捷键组合

---

### 问题 4: 托盘菜单打不开

**可能原因**:
- 右键点击位置不准确
- 系统权限问题

**解决方法**:
- 确保右键点击的是托盘图标
- 尝试以管理员身份运行程序

## 💡 提示

### 如何查看托盘图标

Windows 系统托盘可能在折叠区域：

1. 查看右下角任务栏
2. 点击 `^` 箭头
3. 找到翻译工具图标
4. 可以拖拽到可见区域

### 如何完全退出程序

**重要**: 
- ❌ 点击窗口的 ❌ **不会**退出程序
- ✅ 必须通过托盘菜单 → "退出程序"

### 内存占用

- 隐藏到托盘后仍占用约 95MB 内存
- 这是正常的 Electron 应用行为
- 不需要时可以完全退出
