## 🎨 艺术字样式详解

### CSS 效果分解

```css
.app-title {
  /* 基础样式 */
  font-size: 16px;              /* 大号字体 */
  font-weight: bold;            /* 粗体 */
  color: white;                 /* 白色文字 */
  letter-spacing: 1px;          /* 字间距 */
  
  /* 艺术字特效 */
  text-shadow: 
    2px 2px 4px rgba(0, 0, 0, 0.3),      /* 1. 左下阴影（立体感）*/
    0 0 10px rgba(255, 255, 255, 0.5),   /* 2. 内发光 */
    0 0 20px rgba(255, 255, 255, 0.3);   /* 3. 外发光 */
}
```

### 三层阴影效果

#### 第一层：立体阴影
```css
2px 2px 4px rgba(0, 0, 0, 0.3)
```
- **偏移**: 右下方 2px
- **模糊**: 4px
- **颜色**: 半透明黑色
- **作用**: 创造立体浮雕效果

#### 第二层：内发光
```css
0 0 10px rgba(255, 255, 255, 0.5)
```
- **偏移**: 无（中心发光）
- **模糊**: 10px
- **颜色**: 半透明白色
- **作用**: 文字内部发光

#### 第三层：外发光
```css
0 0 20px rgba(255, 255, 255, 0.3)
```
- **偏移**: 无
- **模糊**: 20px
- **颜色**: 更透明的白色
- **作用**: 文字外部光晕

## 📐 布局设计

### 完整的标题栏结构

**HTML**:
```html
<div class="title-bar">
  <!-- 标题（左侧） -->
  <div class="app-title">
    迷你翻译小工具 By Bluecap
  </div>
  
  <!-- 控制按钮（右侧） -->
  <div class="window-controls">
    <button class="control-btn settings-btn">⚙️</button>
    <button class="control-btn minimize-btn">−</button>
    <button class="control-btn close-btn">×</button>
  </div>
</div>
```

**CSS Flexbox 布局**:
```css
.title-bar {
  display: flex;
  justify-content: space-between;  /* 两端对齐 */
  align-items: center;             /* 垂直居中 */
}

.app-title {
  /* 自动占据左侧空间 */
}

.window-controls {
  /* 按钮组在右侧 */
}
```

### 可拖拽区域保护

**重要设置**:
```css
.title-bar {
  -webkit-app-region: drag;  /* 整个标题栏可拖拽 */
}

.app-title {
  -webkit-app-region: no-drag;  /* 标题文字本身不可拖拽 */
}
```

**原因**:
- ✅ 避免点击文字时触发窗口拖动
- ✅ 保持文字可选中（虽然实际用不到）
- ✅ 符合用户交互习惯

## 💡 设计思路

### 为什么选择艺术字？

1. **品牌识别度** ↑
   - 独特的视觉效果
   - 更容易被记住
   - 提升应用质感

2. **视觉层次** ↑
   - 标题与正文区分明显
   - 引导用户注意力
   - 增强界面深度

3. **美观度** ↑
   - 现代感强
   - 精致细腻
   - 专业印象

### 颜色搭配

**整体配色方案**:
```
标题栏背景：紫色渐变 (#667eea → #764ba2)
标题文字：白色 (#ffffff)
阴影：黑色 + 白色光晕
按钮：半透明白色背景
```

**和谐统一**:
- ✅ 紫色背景 + 白色文字 = 经典搭配
- ✅ 光晕效果呼应渐变背景
- ✅ 整体风格一致
