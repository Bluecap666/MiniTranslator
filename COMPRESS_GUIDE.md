# 🗜️ UPX 压缩指南 - 进一步减小体积

## 📊 压缩效果预期

**压缩前**: ~70MB  
**压缩后**: ~35-40MB  
**压缩率**: 约 50-60%

虽然还是超过 10MB，但已经能节省近一半的空间！

## 🔧 安装 UPX

### 方法 1: 使用 Chocolatey（推荐）

```powershell
# 如果已安装 Chocolatey
choco install upx
```

### 方法 2: 手动安装

1. **下载 UPX**
   - 访问：https://github.com/upx/upx/releases
   - 下载：`upx-4.x.x-win64.zip`

2. **解压文件**
   - 解压到：`C:\Program Files\UPX\`

3. **添加到 PATH**
   ```powershell
   # 以管理员身份运行 PowerShell
   $envPath = [Environment]::GetEnvironmentVariable("Path", "Machine")
   [Environment]::SetEnvironmentVariable("Path", "$envPath;C:\Program Files\UPX", "Machine")
   ```

4. **验证安装**
   ```powershell
   upx --version
   ```

## 🚀 执行压缩

### 方式 1: 使用自动化脚本（推荐）

```powershell
# 在项目根目录运行
.\compress-upx.ps1
```

脚本会自动：
- ✓ 检查 UPX 是否安装
- ✓ 找到打包好的 exe 文件
- ✓ 创建备份
- ✓ 执行最大压缩
- ✓ 显示压缩结果
- ✓ 询问是否删除备份

### 方式 2: 手动压缩

```powershell
# 1. 进入 dist 目录
cd dist

# 2. 执行压缩（最大压缩级别）
upx --best -9 迷你翻译小工具-1.0.0-Portable.exe

# 3. 查看结果
dir *.exe
```

## 📈 压缩参数说明

### UPX 压缩级别

| 参数 | 压缩级别 | 速度 | 推荐场景 |
|------|---------|------|---------|
| `-1` | 最快 | ⚡⚡⚡ | 测试用 |
| `-5` | 平衡 | ⚡⚡ | 日常使用 |
| `-7` | 较好 | ⚡ | 分发版本 |
| `-9` | 最大 | 🐌 | **最终发布** |
| `--best` | 极限 | 🐌🐌 | **追求极致** |

### 推荐配置

```powershell
# 最佳平衡（推荐）
upx --brute 迷你翻译小工具-1.0.0-Portable.exe

# 或者使用极限压缩
upx --ultra-brute 迷你翻译小工具-1.0.0-Portable.exe
```

## ⚠️ 注意事项

### 1. 杀毒软件误报

UPX 压缩的可执行文件可能被误报为病毒。

**解决方案**:
- 添加白名单
- 使用数字签名
- 向杀毒软件厂商提交误报

### 2. 启动速度

压缩后启动速度可能略微变慢（0.1-0.3 秒），因为需要解压。

**影响**: 几乎感觉不到

### 3. 兼容性

UPX 压缩的文件在所有 Windows 版本上都能正常运行。

**验证**: ✅ 完全兼容

## 📝 常见问题

### Q: 压缩后无法运行？
A: 可能是杀毒软件拦截。添加白名单或暂时关闭杀毒软件。

### Q: 能不能压缩到 10MB？
A: 不能。Electron 运行时本身就很大，UPX 只能压缩到 35-40MB。

### Q: 压缩会影响功能吗？
A: 不会。UPX 只是压缩可执行文件，不影响运行时功能。

### Q: 值得压缩吗？
A: 
- ✅ 如果要通过网络分发 → 值得
- ✅ 如果要节省空间 → 值得  
- ❌ 如果只是自己用 → 无所谓
