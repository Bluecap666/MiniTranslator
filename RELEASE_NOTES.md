## ✨ 核心功能

### 🎯 主要特性

- **🚀 多平台并行翻译** - 一次翻译，同时获取 Google、百度、有道等 7+ 个平台结果
- **⚡ 快速响应** - 8 秒超时控制，拒绝漫长等待
- **📌 磁吸式设计** - 窗口始终置顶，不占用过多桌面空间

- **🗑️ 一键清空** - 快速清空输入输出内容

- **💾 智能保存** - 配置信息本地保存，下次启动自动加载

## 🚀 快速开始

### 方式一：直接使用（推荐）⭐

1. 下载 `迷你翻译小工具 -1.0.0-Portable.exe`
2. 放到任意目录（如 `D:\Tools\`）
3. 双击运行即可使用
4. 首次运行时自动生成配置文件

> 💡 **提示**: 配置文件 `config.json` 会自动生成在 exe 同目录下

### 方式二：源码运行

```bash
# 克隆项目
git clone https://github.com/Bluecap666/MiniTranslator.git
cd MiniTranslator

# 安装依赖
npm install

# 启动应用
npm start

# 打包成 EXE
npm run build
```

## 📖 使用说明

### 🎯 基本流程

1. **输入文本** - 在上方输入框输入要翻译的文本
2. **执行翻译** - 点击「翻译」按钮或使用 `Ctrl + Enter`
3. **查看结果** - 翻译结果以列表形式展示，格式：`平台名称：翻译结果`
4. **清空内容** - 点击「🗑️ 清空」按钮一键清空

### ⚙️ API 配置

**第 1 步**：点击主窗口右上角的 ⚙️ 齿轮图标  
**第 2 步**：选择需要配置的服务，填写 API Key / Secret  
**第 3 步**：点击「测试」按钮验证可用性  
**第 4 步**：显示「✓ 可用」后点击「保存配置」

详细的 API 申请指南请查看 [README.md](./README.md)

### 项目结构

```
MiniTranslator/
├── main.js              # Electron 主进程入口
├── preload.js           # 预加载脚本（IPC 通信桥接）

├── styles.css           # 主窗口样式表

├── config.js            # 配置窗口逻辑
├── package.json         # 项目配置和打包配置
└── dist/                # 打包输出目录
    └── 迷你翻译小工具 -1.0.0-Portable.exe
```

## 💡 常见问题

### Q: 为什么翻译结果为空？
**A**: 可能是未配置 API 密钥或网络问题。请打开配置窗口检查并测试 API。

### Q: 配置文件在哪里？
**A**: `config.json` 会自动生成在 exe 同目录下。

### Q: 如何备份配置？
**A**: 直接复制整个文件夹即可备份所有数据。

### Q: 可以只使用某一个翻译服务吗？
**A**: 可以！只需要配置该服务的 API 密钥，其他服务留空即可。

### Q: 为什么有些服务测试失败？
**A**: 可能是 API 密钥错误、余额不足或网络被拦截。
